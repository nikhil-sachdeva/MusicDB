Music Database Project

Node.js is used to create the server and connect it to the music database, created using MySQL. The database comprises of attributes like song name, artist, album, duration, genre, etc and has a provision to store multple songs in playlists, categorised by the user preferences.

Features :

  1. View all songs in the database
  2. Add a song in the database
  3. Add songs to the playlist
  4. View songs of any playlist
  5. Execute queries
    • View all artists
    • View all songs starting with letter 's'
    • View songs with duration less than 5 minutes
